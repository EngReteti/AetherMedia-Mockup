# Aether Media - Visual Identity

## 🎨 Color Palette
| Usage | Hex Code | Preview |
| :--- | :--- | :--- |
| Background (Primary) | `#0F0F0F` | Deep Charcoal |
| Surface (Secondary) | `#1A1A1A` | Dark Grey |
| Accent (Action) | `#007BFF` | Electric Blue |
| Text (High Emphasis) | `#F5F5F5` | Off-White |
| Text (Low Emphasis) | `#A0A0A0` | Muted Grey |

## Typography
- **Headings:** *Montserrat* (Bold/Black) - For a cinematic, heavy feel.
- **Body:** *Inter* (Regular/Medium) - For high legibility on mobile.

## UI Elements
- **Buttons:** 8px Border Radius (Soft-round)
- **Icons:** Minimalist Line Icons (Lucide or Feather style)
